Load and save content using MHT or HTML format

Using the Load from File and Download as File functions you can edit content using external programs, like Microsoft® Word, including rich content like embedded images.

To load content from a Word file:
  1. Open the file in Word
  2. From the File menu, choose 'Save as' and select the file type Single File Web Page (*.mht; *.mhtml)
  3. Select ‘Load from File’ from the Tools menu and select the file saved from the previous step
To save content for editing in Word:
  1. Select ‘Download as File’ from the Tools menu
  2. This should present a file download dialog; save the file to a folder on your computer
  3. Open the downloaded file in Word, or other editor that supports MHT format
  4. Use ‘Load from File’ to load changes back into the online editor
